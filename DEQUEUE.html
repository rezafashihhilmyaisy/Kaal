Struktur Data: Deques

1. Definisi Deques

  • Pengertian: Deques (Double-Ended Queues) merupakan struktur data ketiga setelah stacks dan queues. Perbedaan utamanya terletak pada fleksibilitas manipulasi ujung data.
  • Perbandingan Ujung Operasi:
    • Stacks: Penambahan dan penghapusan data dilakukan pada ujung yang sama (top).
    • Queues: Penambahan data dilakukan di satu ujung (rear) dan penghapusan di ujung berbeda (front).
    • Deques: Penambahan maupun penghapusan data dapat dilakukan secara bebas di kedua ujung, baik dari ujung front maupun ujung rear.

2. Operasi Dasar pada Deques

Beberapa fungsi atau operasi dasar yang berlaku pada Deques meliputi:

  • deque(): Inisialisasi struktur data deque baru dalam keadaan kosong.
  • addFront(data): Menambahkan data baru di ujung depan (front).
  • addRear(data): Menambahkan data baru di ujung belakang (rear).
  • removeFront(): Menghapus dan mengambil data pada ujung depan (front).
  • removeRear(): Menghapus dan mengambil data pada ujung belakang (rear).
  • isEmpty(): Memeriksa apakah deque dalam kondisi kosong.
  • size(): Mengembalikan nilai berupa jumlah total data di dalam deque.

3. Representasi Deques Menggunakan List Python

Struktur data ini direpresentasikan menggunakan tipe data List bawaan Python dengan ketentuan indeks sebagai berikut:

  • Indeks Awal (Indeks ke-0): Berperan sebagai posisi depan (front).
    • Penambahan data dilakukan menggunakan insert(0, data).
    • Penghapusan data dilakukan menggunakan pop(0).
  • Indeks Akhir (Indeks Terakhir): Berperan sebagai posisi belakang (rear).
    • Penambahan data dilakukan menggunakan append(data).
    • Penghapusan data dilakukan menggunakan pop() tanpa argumen indeks.

4. Contoh Implementasi Kasus: Pengecekan Palindrom

  • Definisi: Palindrom adalah rangkaian kata atau bilangan yang memiliki urutan karakter yang terbaca sama, baik ketika dibaca dari depan maupun dari belakang.
  • Pemanfaatan Deque: Deques sangat tepat digunakan untuk masalah ini. Cara kerjanya adalah dengan memasukkan semua karakter kata ke dalam deque, lalu membandingkan karakter pada ujung front dan ujung rear secara bersamaan. Jika karakter pada kedua ujung tersebut selalu sama hingga tersisa 1 atau 0 karakter, maka kata tersebut divalidasi sebagai palindrom.
