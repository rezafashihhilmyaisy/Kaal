REKURSIF

1. Pengantar Pemrograman Rekursif

  • Beberapa permasalahan tertentu di dalam pemrograman membutuhkan penyelesaian dengan cara perulangan, seperti masalah pengurutan data, perhitungan deret, dan sebagainya.
  • Terdapat dua metode utama yang dapat digunakan untuk menyelesaikan permasalahan perulangan tersebut, yaitu:
    1. Iteratif: Penyelesaian menggunakan struktur perulangan konvensional dengan syntax for atau while.
    2. Rekursif: Teknik pemrograman yang memecah masalah besar menjadi sub-masalah dengan ukuran yang semakin kecil agar lebih mudah dipecahkan.

2. Karakteristik dan Aturan Rekursif

  • Teknik rekursif dilakukan dengan cara memanggil fungsi itu sendiri di dalam tubuh fungsi tersebut.
  • Setiap pemanggilan rekursif harus melibatkan argumen atau parameter yang berukuran lebih kecil dari state sebelumnya.
  • Terdapat tiga aturan penting yang wajib dipenuhi dalam teknik rekursif:
    • Memiliki Base Case: Kondisi dasar yang berfungsi untuk menghentikan pemanggilan fungsi secara terus-menerus (mencegah infinite loop).
    • Mengubah State: Fungsi harus memiliki sintaks yang mampu mengubah status permasalahan, sehingga seiring berjalan waktu solusi akan bergerak mendekati base case.
    • Memanggil Diri Sendiri: Fungsi harus memanggil dirinya sendiri.

3. Contoh Kasus Eksperimen Rekursif

A. Perhitungan Faktorial

  • Pola matematika faktorial didefinisikan sebagai n! = n × (n-1) × (n-2) × ... × 1.
  • Secara rekursif, faktorial dapat dirumuskan menjadi: n! = n × (n-1)! dengan base case 1! = 1.

B. Triangular Numbers

  • Triangular numbers merupakan deret bilangan yang membentuk susunan kotak/titik menyerupai segitiga: 1, 3, 6, 10, 15, 21, 28, 36, ....
  • Untuk menentukan deret ke-n, kita menjumlahkan bilangan dari n mundur hingga 1. Contohnya pada deret ke-4 (n=4), jumlah totalnya adalah 4 + 3 + 2 + 1 = 10. Base case dari masalah ini tercapai ketika deret ke-1 bernilai 1.

C. Konversi Bilangan (Desimal ke Base Lain)

  • Bilangan integer standar menggunakan basis 10 (desimal) dengan karakter angka 0 s.d 9. Basis 2 (biner) hanya menggunakan angka 0 dan 1. Basis 8 (oktal) menggunakan 0 s.d 7. Basis 16 (heksadesimal) menggunakan kombinasi angka 0 s.d 9 serta huruf A sampai F.
  • Proses konversi dari desimal ke basis lain memanfaatkan operasi pembagian bulat (//) dan sisa bagi atau modulo (%) secara berulang.

D. Towers of Hanoi

  • Terinspirasi dari mitos sebuah kuil di India di mana pendeta diperintahkan memindahkan 64 piringan emas dengan ukuran berbeda-beda dari tiang asal ke tiang tujuan lewat tiang bantuan.
  • Aturan pemindahan piringan:
    • Piringan harus dipindahkan satu demi satu.
    • Piringan berukuran lebih kecil harus selalu berada di atas piringan yang lebih besar.
  • Algoritma Rekursif Hanoi untuk n piringan:
    1. Pindahkan piringan sebanyak n-1 dari tiang awal ke tiang bantuan melalui tiang tujuan.
    2. Pindahkan piringan terbesar (ke-n) langsung dari tiang awal ke tiang tujuan.
    3. Pindahkan kembali piringan sebanyak n-1 dari tiang bantuan ke tiang tujuan melalui tiang awal.

4. Visualisasi Grafik secara Rekursif (Modul Turtle & Fraktal)

  • Rekursif dapat dimanfaatkan untuk keperluan visualisasi gambar menggunakan modul Turtle pada Python.
  • Modul Turtle diumpamakan sebagai seekor kura-kura pembawa pena. Jika posisi ekornya turun (down), ia akan menggambar pola seiring pergerakannya, sedangkan jika ekornya naik (up), ia tidak akan menggambar.
  • Fraktal: Cabang matematika yang memiliki kemiripan dengan rekursif, di mana terdapat suatu bentuk geometri dasar yang dikembangkan secara berulang terus-menerus. Contoh fraktal yang dibahas adalah struktur bentuk pohon (tree) dan Segitiga Sierpinski (segitiga yang di dalamnya melahirkan fraktal empat buah segitiga baru dengan ukuran sama secara berulang).

def triangularNumbers1(n): total = 0 temp = n while temp >= 1: total += temp temp -= 1 return total print(triangularNumbers1(2)) # Output: 3

4. Perhitungan Triangular Numbers (Rekursif)

def triangularNumbers2(n): if n == 1: # Base Case return 1 else: return n + triangularNumbers2(n - 1) # Rekursi print(triangularNumbers2(8)) # Output: 36

def konversiDesimal(bilangan, base): charBilangan = '0123456789ABCDEF' if bilangan < base: # Base Case return charBilangan[bilangan] else: temp = bilangan // base ind = bilangan % base return konversiDesimal(temp, base) + charBilangan[ind] print(konversiDesimal(5, 2)) # Output: 101 (Biner)
