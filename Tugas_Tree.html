Penjelasan


    
1. Token

    

        Token adalah unit atau bagian terkecil dari input yang dibaca oleh parser.
        Token dapat berupa angka, variabel, operator, maupun tanda kurung.
    


    
Contoh ekspresi:

    
a + b


    
Token yang dihasilkan:

    
['a', '+', 'b']


    

        Pada proses parsing, token dibaca satu per satu untuk membangun struktur pohon (tree).
    


    
2. Current Node

    

        Current Node adalah node yang sedang aktif atau sedang diproses oleh parser
        saat membangun Parse Tree.
    


    
Contoh:

    
a + b


    

        Ketika parser membaca token "+", maka current node berpindah ke node operator "+".
        Saat membaca "a" atau "b", current node berada pada node yang menyimpan nilai tersebut.
    


    

        Current Node digunakan untuk menentukan posisi node baru yang akan ditambahkan ke dalam tree.
    


    
3. Parse Tree

    

        Parse Tree adalah struktur pohon yang menggambarkan hubungan antara operator dan operand
        dalam suatu ekspresi.
